Beginner Animation Samples

I created the following CSS animations based on a tutorial I found in this article.

In addition to the tutorial, I incorporated the use of a random image generator API called PicSum to dynamically display photos, eliminating the need for static image files. This approach not only adds a dynamic and engaging element to the project but also reduces the need for manual image management and maintenance. I also dynamically add/remove a spinner to indicate that the page is loading while the API retrieves its data.
